## 快速开始

</div>

```bash
# 克隆项目
git clone https://github.com/Otter-Knight/paper-pulse.git
cd paper-pulse

# 安装依赖
npm install

# 复制环境配置
cp .env.example .env

# 启动开发服务器
npm run dev
```
